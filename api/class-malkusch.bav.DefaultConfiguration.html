Class DefaultConfiguration

Default configuration uses malkusch\bav\FileDataBackendContainer and any available UTF-8 encoder.

Automatic installation is enabled.

The update plan is set to malkusch\bav\LogUpdatePlan

If no UTF-8 encoding is supported ISO-8859-15 will be used.
